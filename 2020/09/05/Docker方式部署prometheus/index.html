构建镜像并运行

  • -h prometheus139-210：设定容器的hostname为prometheus139-210；
  • -m 8g：设定容器最大使用内存为8G；
  • --cpuset-cpus=28-31：制定容器运行在编号为28到31的CPU上；
  • --name=prometheus139-210：制定容器的名字；
  • --network：设定容器使用的网络；
  • --ip：容器固定IP；
  • -v：挂在数据卷到容器中作为数据目录；
  • -p：端口映射；
