git init会创建一个.git的隐藏目录，其中的文件和用途如下：

  • hooks：存放一些脚本，可设定指定的git命令运行后执行相应的脚本；
  • info：包含仓库的信息；
  • logs：保存所有更新的引用记录；
  • objects：项目中的所有对象；
  • refs：保存指向对象的索引；
  • COMMIT_EDITMSG：保存最新的commit信息；
  • config：git仓库的配置文件；
  • description：仓库描述信息；
  • index：二进制格式文件，暂存区；
  • HEAD：包含了当前分支的引用，通过这个文件git可以得到下一次commit的parent；
  • ORIG_HEAD：head指针的前一个状态；
